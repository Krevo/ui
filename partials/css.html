Utility CSS classes

pull-left add a float left to the current element
pull-right add a float right to the current element
clear add a clear both to the current element
block-content add a padding
space-top, space-bottom, space-left, space-right For adding paddings
pointer add a cursor pointer to the current element

Tables

You can add a color to table's header by using one of these classes : table-one, table-two et table-three.
